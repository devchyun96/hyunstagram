## 사용 기술

**1. backend**

    언어 : Java
    프레임워크 : spring boot 3.2.1
    라이브러리 : data-jpa / web / websocket / thymeleaf / devtool / lombok / mysql
    빌드 : gradle 8.5
    데이터베이스 : mysql

**2. frontend**
    
    언어: JavaScript
    프레임워크 : node.js 21.6.0
    라이브러리 : react 18.2.0
